Fit a PLDS model using subspace identification (SSID)

Fit PLDS model by subspace identification (SSID). System matrices (A,B,C) are fit by SSID assuming gaussian observations. Iteratively refits output function parameters (C, d) by Poisson MLE using Newton's method.

Parameters
uinputs
n{ parameter_description }
dtsample period
nXnumber of states
nHblock-Hankel data matrix size
d0previous estimate of output bias (d)
wtchosen weighting for SSID singular value \ decomposition (NONE, MOESP, CVA)
wtG0chosen weighting for minimizing model error at \ DC
t0start times for each trial of input/output data
t_startSSIDstart time for the data used in SSID
t_stopSSIDstop time for the data used in SSID
assumeGaussian[NONFUNCTIONAL] whether to assume Gaussian observation model and fit by linear SSID first, then fit output nonlinearity by Poisson MLE
zmeasurements
