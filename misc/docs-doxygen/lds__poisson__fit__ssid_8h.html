Detailed Description

PLDS SSID fit type.

This file declares and partially defines a type by which Poisson-output LDS models are fit by a subspace identification (SSID) algorithm (lds::gaussian::ssidFit_t). Currently, these models are fit assuming linear/Gaussian output, and then an exponential output is fit by maximum likelihood under the Poisson assumption. Future work should implement SSID after Poisson<->Gaussian moment conversions as outlined in [2].

References: [1] van Overschee P, de Moore B. (1996) Subspace Identification for Linear Systems. Boston: Springer. [2] Buesing L, Macke JH, Sahani M. (2012) Spectral learning of linear dynamics from generalised-linear observations with application to neural population data. NIPS 25.
